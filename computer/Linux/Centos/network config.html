WIFI ​

1. 安装NetworkManager ​

sh
sudo yum install NetworkManager
1

2. 启动并启用NetworkManager服务 ​

sh
sudo systemctl start NetworkManager
sudo systemctl enable NetworkManager
1
2

3. 查找可用的WiFi网络 ​

sh
nmcli dev wifi list
1

4. 创建WiFi连接 ​

sh
nmcli dev wifi connect <MyHomeNetwork> password <MyPassword>
1

5. 检查连接状态 ​

sh
nmcli connection show
1

6. 激活连接 ​

如果你的设备还没有自动连接到网络，你可以手动激活连接：

sh
nmcli connection up id <MyHomeNetwork>
1

7. 确认网络连接 ​

sh
ping baidu.com -c 4
1
