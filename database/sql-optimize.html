设置数据读取级别 ​

  • 顺序读serializable
  • 重复读repeatable read
  • 读已提交read committed
  • 读未提交read uncommited
sql
# 设置读取隔离级别
set session transaction isolation level repeatable read;

#回滚
rollback

#提交

blocking_query as 'running sql',
wait_age as 'blocked time',
sql_kill_blocking_query as 'info'
