C 语言输入输出函数——程序与世界的“对话窗口” 试讲逐字稿

(总时长：约 14 分 30 秒)

(走上讲台，站稳，面带微笑，目光扫视全场)

各位评委老师，大家好！我是今天的 X 号考生。 今天我试讲的题目是《C 语言输入输出函数——程序与世界的“对话窗口”》。下面开始我的试讲。

同学们，大家好！请坐。

一、 结合生活实际导入 (约 1 分 30 秒)

(伸出双手，做出一问一答的手势) 同学们，人与人之间要交流，离不开“提问”和“回答”。比如，我问大家：“今天天气怎么样？”大家可能会回答：“晴天！”或者“下雨了！”。这一问一答，就完成了一次信息的传递。

那么，我们编写的程序，它是一个在电脑里运行的小世界。我们如何与这个“小世界”交流呢？我们如何告诉它我们的想法，它又如何把结果告诉我们呢？

(稍作停顿，指向电脑屏幕)

大家看，这个黑色的命令行窗口，它就是程序与外界交流的“桥梁”。程序通过它来“听”我们说话，也通过它来“说”话给我们听。而实现这种“听”和“说”功能的神奇工具，就是我们今天要学习的——输入输出函数。它们就像是程序为我们打开的两个“对话窗口”，一个负责输入(Input)，一个负责输出(Output)。今天，我们就来学习其中最常用、最重要的两位主角：printf 和 scanf。

二、 新授、学生互动与巩固练习 (约 10 分钟)

  1. 输出函数 printf —— 程序的“嘴巴”

(转身板书：printf)

printf 函数，读作“print-f”，它的作用是程序的“嘴巴”，负责把程序运行的结果输出到屏幕上，告诉我们。

它的基本格式是：printf("格式控制字符串", 输出列表);

(通过板书和举例，逐步讲解)

简单“说话”：printf("Hello, World!"); // 程序会直接在屏幕上“说”出：Hello, World!

带变量的“说话”：如果程序里有一个变量 int score = 95;，我们想把它说出来，该怎么办？ 这时就需要格式控制符，它像一个占位符，告诉程序：“嘿，这里要放一个整数进来。” printf("我的成绩是：%d 分", score); // 程序会输出：我的成绩是：95 分

(强调) 这里的 %d 就是专门为整数(int) 准备的占位符。

常用格式控制符家族：

%d — 用于输出整数 (decimal)

%f — 用于输出实数 (float)，默认保留 6 位小数。

%c — 用于输出单个字符 (character)

%s — 用于输出字符串 (string)

【互动环节一：printf 填空游戏】 (PPT 展示代码和预期输出) 假设我们有：int age = 16; char grade = 'A'; float height = 1.75; 请填空，让程序能输出：“我今年 16 岁，成绩是 A 等，身高是 1.75 米。”

printf("我今年_**_岁，成绩是__**等，身高是____米。", age, grade, height);

(给学生 30 秒思考，请一位同学回答) (教师回应) 非常好！正确答案是：%d, %c, %f。恭喜你，已经掌握了让程序“开口说话”的秘诀！

  1. 输入函数 scanf —— 程序的“耳朵”

(转身板书：scanf)

光有“嘴巴”还不够，程序还得有“耳朵”来听我们说话。scanf 函数，读作“scan-f”，就是程序的“耳朵”，负责从键盘读取我们输入的数据。

它的基本格式是：scanf("格式控制字符串", &变量名);

(通过对比和举例，重点讲解)

与 printf 对比：printf("我的成绩是：%d", score); // 把变量 score 的值显示出来 scanf("%d", &score); // 从键盘读入一个整数，存到变量 score 里

关键符号 & (取地址符)： (生活化比喻) 这个&符号非常重要！它就像是告诉快递员你家的具体门牌号。如果没有这个门牌号，快递员（输入的数据）就不知道把包裹（数据）送到哪里去，会导致程序出错！所以，&千万不能丢（除了后面学到的数组名等特殊情况）。

实战演示：我们来写一个完整的对话程序。 (在黑板或 PPT 上写代码)

c #include <stdio.h> int main() { int number; printf("请输入一个数字："); // 程序先“问”我们 scanf("%d", &number); // 程序“听”我们输入 printf("你输入的数字是：%d\n", number); // 程序再“回答”我们 return 0; } 【互动环节二：纠错与巩固练习】 (PPT 展示一段有问题的代码)

c #include <stdio.h> int main() { float price; printf("请输入商品价格："); scanf("%f", price); // 这里有一个常见错误！ printf("价格是：%f 元\n", price); return 0; } 请同学们火眼金睛，找出这段代码中的错误，并说明为什么。 (引导学生发现并回答：scanf 行中，price 变量前缺少了取地址符&)

(教师总结) 对！这是一个非常典型的错误。记住，scanf 中，除了字符串，变量前面一定要加&，把“门牌号”告诉程序。

三、 课堂总结与升华 (约 1 分钟)

同学们，今天我们学习了 C 语言中两个至关重要的“对话窗口”：printf 和 scanf。printf 是程序的“嘴巴”，负责输出，告诉我们结果；scanf 是程序的“耳朵”，负责输入，聆听我们的指令。

大家想一想，从今天起，我们编写的程序就不再是自说自话的“哑巴”程序了，它终于可以和我们互动了！这标志着我们向编写实用软件迈出了一大步。无论是未来的“学生成绩管理系统”，还是“简易计算器”，都离不开我们今天学习的这一“听”一“说”的完美配合。掌握了它们，就相当于为我们程序的智慧，插上了与外界沟通的翅膀！

四、 布置作业 (约 1 分钟)

今天的课后作业是：

基础题：编写一个“个人信息登记”程序。要求程序能提示用户输入姓名（字符串暂用%s）、年龄和身高，并能将输入的信息完整地打印出来。

挑战题：尝试编写一个简易的“加法计算器”程序。让用户先后输入两个整数，程序然后计算出它们的和并输出。格式如：“5 + 3 = 8”。

希望同学们亲手敲写代码，感受与程序“对话”的乐趣。下课！

(面向评委，鞠躬) 各位评委老师，我的试讲到此结束，感谢各位老师的聆听！
